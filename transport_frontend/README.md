# Trust Drive India Frontend

Landing page for `Trust Drive India`, built with React, TypeScript, and Vite.

## Contact Number Setup

Create a `.env` file in the project root and add:

```bash
VITE_CONTACT_NUMBER=+919000000000
```

This number is used for:

- `Call now` dial links
- `WhatsApp` launcher with a pre-generated booking message

If no environment value is set, the UI falls back to a placeholder number.

## Run the Project

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
